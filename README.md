# Sample_Projects

This repo contains sample projects for various programming languages and frameworks. Each project is designed to demonstrate best practices and common patterns in software development.

## Projects Included

### Machine Learning

- [**PCA + Random Forest Classificator**](sklearn/[BASIC]%20PCA_RandForestPipeline/): A project that combines Principal Component Analysis (PCA) for dimensionality reduction with a Random Forest Classifier for classification tasks as a ML Pipeline.

- [**XGBoost Classificator**](xgboost/[BASIC]%20classification_iris/): A simple project that showcases the XGBoost algorithm for classification tasks.

- [**GCP ML API**](cloud/[BASIC]%20GCP_ML_API/): A project that demonstrates how to use Google Cloud Platform's Machine Learning API for building and deploying machine learning models.

## How to Use

Each project contains a README file with instructions on how to set up and run the code.

Also, every project contains a `requirements.txt` file that lists the necessary dependencies.

As an example, for the PCA + Random Forest Classificator project:

```bash
cd "sklearn/[BASIC] PCA_RandForestPipeline/"
pip install -r requirements.txt
```
